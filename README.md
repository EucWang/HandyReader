## Features

- Support for EPUB/MOBI/AZW/AZW3/FB2/TXT/MD/PDF/Mp3 format
- Material You design
- Color picker functionality
- Room database integration for local storage
- Jetpack Compose UI
- Hilt dependency injection
- Coil for image loading
- Datastore for preferences
- Paging support
